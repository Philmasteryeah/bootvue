
# bootvue
- Spring Boot 2
- Vue.js
- AdminLTE

# How to start

```
$ git clone https://github.com/Philmasteryeah/bootvue.git
$ cd bootvue
$
$ mvn clean install
$ mvn --projects backend spring-boot:run
$
```
- open http://localhost:8098/

- only frontend

```
$
$ cd bootvue\frontend\
$
$ npm start serve
$
```
- http://localhost:8081/
