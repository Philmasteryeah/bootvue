
# bootvue
- Spring Boot 2
- Vue.js
- CoreUI with AdminLTE
- axios for REST

# How to start
- start both

```
$ git clone https://github.com/Philmasteryeah/bootvue.git
$ cd bootvue
$
$ mvn clean install
$ mvn --projects backend spring-boot:run
$
```
- open http://localhost:8098/

- or only the frontend

```
$
$ cd bootvue\frontend\
$
$ npm start serve
$
```
- http://localhost:8081/
